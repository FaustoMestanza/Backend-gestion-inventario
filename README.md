# Inventory Microservice

## Description
This microservice manages inventory operations within a distributed system.

## Technologies
- Django REST Framework
- PostgreSQL
- Docker

## Features
- CRUD operations
- JWT authentication
- API endpoints

## Architecture
Part of a microservices-based system for inventory and loan management.
